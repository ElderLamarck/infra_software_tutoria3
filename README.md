# Infra_software_3
Atividade para obtenção de nota parcial da cadeira "Infraestrutura de Software"

## Como clonar o repositório:
```bash
$ git clone https://github.com/ElderLamarck/infra_software_tutoria3.git
$ cd infra_software_tutoria3
```

## Como utilizar o programa:
O diretorio possui um arquivo makefile, nele existem os comandos run e clean, para compilar é necessario usar o "make" e em seguida "make run" para rodar. O programa recebe uma matrix e cria threads para somar os valores de cada linha e guarda em uma variavel global
